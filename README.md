# JDBC-WRAP

- Java JDBC 를 Hibernate 처럼 학습용으로 Wrap 합니다.

# How To Work

- Issue 를 먼져 작성합니다.
- Git Commite Prefix 에 `[Issue Number] commit summary` 로 작성합니다.
- Pull Request 를 날립니다.
- Review 후 머지 됩니다.

# Rules

- Issue 에 관련된 내용은 Issue 에 모두 정리합니다.
- 중복된 Issue 가 있다면 Close 하고 연관된 Issue 에 대한 링크를 남기고 하나의 이슈에서 최대한 토론합니다.
